# cmsc191-2s-ay2016-2017-twitter-data
Github for CMSC 191 exercise on collecting data from Twitter using R

Tweets contain the keywords: Trump, Donald Trump, POTUS, WelcomeToCanada, RefugeesWelcome, MuslimBan

Each file contains approximately 300k rows of data.
Tweets were separated into smaller files because of Github's 100.00 MB file size limit.
